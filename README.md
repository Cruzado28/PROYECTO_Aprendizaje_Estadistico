**PROYECTO DE APRENDIZAJE ESTADÍSTICO**

**MODELOS ESTADÍSTICOS PARA LA DETECCIÓN Y PREVENCIÓN DE ATAQUES DE PHISHING EN ENTORNOS DIGITALES**

El presente proyecto se enmarca en la intersección entre la ciberseguridad y el aprendizaje estadístico supervisado, con el objetivo de diseñar modelos capaces de detectar y prevenir ataques de phishing en entornos digitales. Para ello, se consideran atributos técnicos y textuales como la estructura de URLs, la validez de certificados SSL, el tráfico web, metadatos de correos y patrones lingüísticos. La propuesta se sustenta en el uso de algoritmos de clasificación supervisada como árboles de decisión (J48), regresión logística, k-NN (IBk) y SVM (SMO), complementados con técnicas de minería de datos y reducción de dimensionalidad (CfsSubsetEval, PCA). De este modo, se busca construir un sistema escalable y eficiente que contribuya a la prevención del fraude digital y mejore la seguridad de usuarios y organizaciones.
